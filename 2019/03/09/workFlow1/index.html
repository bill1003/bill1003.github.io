高手的一流工作術-(1)-Google alias

2019-03-09
工作術, 成長策略, 效率, 高手

常常電腦用沒多久就覺得累了嗎？知道這些小撇步讓你效率level up!

不知道你是否有這種經驗，某個網站沒多久前才去過，但就是忘記擺到哪裡？之前明明有好好放進某個chrom書籤，但現在連放進那個書籤資料夾都忘了？其實對於常用的網頁有一個更快開啟與儲存的妙招，這個妙招可以幫助你在三秒內高效且確實的快速訪問，而這個技巧的名字就叫做

Google自定義搜尋引擎-Google Alias Search Engine

只要事先設定好，在網址欄內打入關鍵字加enter，你就會飛速的跳轉到那個網頁！不要小看優化的這一小步！當你有很多網頁需要來回切換時，這個小妙招會幫你非常大的忙！

googleAlias最大的好處在於

  • 不需要啟動腦袋及眼睛的搜索功能

  • 三秒內直覺開啟，節省找的時間

  • 不用再煩惱重要網站要存在那個書籤，用關鍵字直接訪問就可以！

具體怎麼用？

到chrom左上角的「設定」->管理搜尋引擎->新增

  • 新增搜尋引擎: _ALIAS

  • 關鍵字: 你想要快速訪問這個網站的關鍵字

  • 網址：這個網站的網址

設定完後按新增，然後到網址欄輸入你剛剛定義的關鍵字，這時你就會發現chrome直接開啟你要的網站！是不是很便利呢！

注意事項

注意！關鍵字的定義一定要夠直覺！不然就失去設定alias的意義！最好是你一想到這個網站，心中直覺浮出的第一個英文單字，而那個第一直覺的單字就是你的最佳解關鍵字！若定義的太難太刻意的話，保證你下次也想不起來！那就失去設定alias的意義了！

還在等什麼？快動手設定屬於你自己alias引擎吧！相信你不久後也會對這樣的高效率開網頁感到上癮的！
